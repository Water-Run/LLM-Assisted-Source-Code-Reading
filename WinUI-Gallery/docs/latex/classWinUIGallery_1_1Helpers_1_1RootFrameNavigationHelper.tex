\doxysubsection{Detailed Description}
\doxylink{classWinUIGallery_1_1Helpers_1_1RootFrameNavigationHelper}{Root\+Frame\+Navigation\+Helper} registers for standard mouse and keyboard shortcuts used to go back and forward. There should be only one \doxylink{classWinUIGallery_1_1Helpers_1_1RootFrameNavigationHelper}{Root\+Frame\+Navigation\+Helper} per view, and it should be associated with the root frame.
