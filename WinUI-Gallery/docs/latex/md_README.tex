\doxysection*{Win\+UI 3 Gallery}



Companion app for \href{https://docs.microsoft.com/windows/apps/winui}{\texttt{Win\+UI}} \& \href{https://github.com/microsoft/WindowsAppSDK}{\texttt{Windows App SDK}} APIs

This app demonstrates all of the Win\+UI 3 controls and styles available to make a modern Windows app with the Windows App SDK. It is the interactive companion to the \href{https://docs.microsoft.com/windows/apps/design/basics/}{\texttt{Fluent Design guidelines}} and shows the usage of \href{https://docs.microsoft.com/windows/apps/winui/}{\texttt{Win\+UI}} through interactive samples, tools and code snippets.

\begin{DoxyItemize}
\item {\bfseries{Win\+UI controls samples\+:}} each control page shows the markup and codebehind used to create each example.
\item {\bfseries{Use the Microsoft.\+UI.\+Xaml (Win\+UI) library\+:}} the app includes the latest Win\+UI Nu\+Get package and shows how to use the \href{https://docs.microsoft.com/windows/apps/winui/}{\texttt{Win\+UI}} controls like Navigation\+View, Swipe\+Control, and more.
\item {\bfseries{Adaptive UI\+:}} in addition to showing how each control responds to different form factors, the app itself is responsive and shows various methods for achieving adaptive UI.
\item {\bfseries{Design \& accessibility guidance}}\+: design and accessibility pages help make the gallery a useful developer companion app.
