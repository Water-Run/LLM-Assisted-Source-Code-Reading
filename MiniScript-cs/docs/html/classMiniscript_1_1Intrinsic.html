Classes

class  Result
 Result represents the result of an intrinsic call. An intrinsic will either be done with its work, or not yet done (e.g. because it's waiting for something). If it's done, set done=true, and store the result Value in result. If it's not done, set done=false, and store any partial result in result (and then your intrinsic will get invoked with this Result passed in as partialResult). More...
